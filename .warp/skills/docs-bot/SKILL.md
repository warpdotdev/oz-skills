# Documentation Bot

> **TODO**: This skill is under development.

This skill will teach agents how to automatically update documentation when code is modified, following patterns similar to [Mintlify's automated documentation approach](https://www.mintlify.com/docs/guides/automate-agent).

## Planned Capabilities

When complete, this skill will guide agents to:

- Detect code changes that require documentation updates
- Identify which documentation files need to be modified
- Generate accurate documentation updates based on code changes
- Create pull requests with documentation updates
- Follow documentation style and formatting conventions

## Future Implementation

This skill will be fleshed out with:

- Step-by-step workflow for documentation updates
- Examples of good documentation patterns
